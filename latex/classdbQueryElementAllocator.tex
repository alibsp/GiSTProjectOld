\doxysubsection{Detailed Description}
Allocator of query elements. This class links deallocated elements in the list of free elements, and reused them in future. So number of system memory allocator invocations is dramatically reduced. Cleanup of free elements lst is performed by {\ttfamily db\+Database\+::cleanup()} method
